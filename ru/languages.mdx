---
title: "Поддерживаемые языки"
description: "Узнайте, какие языки поддерживаются API Nexara."
---

API Nexara для распознавания речи поддерживает автоматическое определение языка и транскрибацию для широкого спектра языков. Вы также можете явно указать язык, используя параметр `language` в ваших API-запросах. Для получения дополнительной информации обратитесь к [документации API](/ru/api-reference/endpoint/transcription#body-language).

Ниже приведен полный список поддерживаемых языков с их кодами.
