---
title: "Supported Languages"
description: "Learn which languages are supported by the Nexara API."
---

The Nexara Speech-to-Text API supports automatic language detection and transcription for a wide range of languages. You can also explicitly specify the language using the `language` parameter in your API requests. Refer to the [API Documentation](/en/api-reference/endpoint/transcription#body-language) for more details.

Below is the comprehensive list of supported languages along with their corresponding codes.
